# Contributing to the Benicia Historic Homes Wiki

## Getting Started

1. Visit the wiki at http://localhost:8080 (or wherever it's hosted)
2. Create an account
3. Browse [Category:Stub Pages](http://localhost:8080/wiki/Category:Stub_Pages) to find properties that need research

## What You Can Do

### Add Property Information
- Find the wiki page for an address you want to research
- Click "Edit with form" to use the structured data entry form
- Add: year built, architectural style, historic status, etc.
- Always cite your source

### Transcribe Census Data
- Visit FamilySearch.org to find Benicia census pages
- Use the Census Entry form to add structured records
- Link records to property pages when addresses match
- See `CENSUS_GUIDE.md` for detailed instructions

### Document Sanborn Maps
- View Sanborn map images at the Library of Congress
- Record which streets and blocks appear on each sheet
- Link property pages to the sheets they appear on

### Upload Photographs
- Current photos of historic buildings
- Historical photographs (verify copyright)
- Show the full front facade and architectural details

## Data Quality Standards

- **Cite everything.** Every fact should trace to a source.
- **Don't guess.** Leave fields blank rather than entering uncertain data.
- **Mark uncertainty.** Use the Notes section for tentative conclusions.
- **Use the Data Confidence field.** Stub → Low → Medium → High.
- **Record contradictions.** If sources disagree, document both.

## Editing Guidelines

- Use the forms when available (they enforce consistent structure)
- For free-text sections, use MediaWiki markup
- Don't delete others' work; add your alternative interpretation
- Use category tags for data quality: `[[Category:Needs Review]]`, `[[Category:Needs Address Verification]]`

## Source Citation

Every claim should be sourced. Use the Sources section at the bottom of each page:

```
== Sources ==
* 1910 US Census, Solano County, California, ED 151, Page 5A, Line 32 (FamilySearch)
* 1913 Sanborn Fire Insurance Map, Sheet 5
* DPR 523 Survey Form, Carol Roland, Ph.D., 2009
```
