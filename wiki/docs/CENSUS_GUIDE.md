## Where to Find Records

- **FamilySearch.org** (free) - Browse by state > county > city
- **Ancestry.com** (subscription) - Searchable by name and location
- **NARA** (National Archives) - Original microfilm

## Transcription Workflow

1. Navigate to a Benicia enumeration district page
2. Record the metadata: ED number, page number, line number
3. Transcribe exactly what's written (don't correct spelling)
4. Enter data using `Form:CensusEntry` on the wiki
5. Link to the modern property page if you can identify the address
6. Create Person pages for heads of household

## Page Naming Convention

- With address: `Census 1910 - 123 West G Street - Smith`
- Without address: `Census 1860 - ED1 Page 5 Line 12 - Smith`

## Common Challenges

- **Handwriting**: Use `[?]` for uncertain readings
- **Name spelling**: Census takers spelled phonetically
- **Address changes**: 1910 address may not match modern numbering
- **Missing 1890**: The gap between 1880 and 1900 is unavoidable

## Recommended Order

Start with 1910 (best alignment with 1913 Sanborn), then expand:
1910 → 1920 → 1900 → 1930 → 1940 → 1880 → 1950 → 1870 → 1860 → 1850
